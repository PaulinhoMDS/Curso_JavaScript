Repositório relacionado ao Curso De javaScript, realizado pela plataforma UDEMY.
este curso foi realizado através das aulas do professor André Iacono.

O curso tem como foco: 
- aprender na prática a linguagem JavaScript, do zero!
- entender como funcionam as variáveis, if/else, operadores, lógica booleana, funções, arrays, objetos, loops, strings, e etc.
- Entender como os desenvolvedores Javascript trabalham no dia a dia
- Praticar  habilidades realizando os diversos projetos do curso
- Prepare-se para o mercado de trabalho entendendo como o JavaScript realmente funciona!
- Aprender tudo sobre POO (programação orientada a objetos) com JavaScript, trabalhando com protótipos e classes
